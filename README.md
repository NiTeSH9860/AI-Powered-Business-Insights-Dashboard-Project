# AI-Powered Business Insights Dashboard

A production-ready Flask application that provides AI-driven business intelligence with real-time analytics, predictive forecasting, and smart recommendations.

## Features

- **Real-time Dashboard**: Interactive metrics monitoring with automatic updates
- **AI-Powered Insights**: GPT-4 powered plain-English explanations of business trends
- **Predictive Analytics**: Sales forecasting using Prophet and XGBoost
- **Churn Prediction**: ML-based customer churn risk assessment
- **Actionable Recommendations**: AI-generated business strategies
- **Interactive Visualizations**: Plotly charts with drill-down capabilities

## Tech Stack

- **Backend**: Flask, SQLAlchemy, Python
- **AI/ML**: OpenAI GPT-4, Prophet, XGBoost, Scikit-learn
- **Data Processing**: Pandas, NumPy
- **Frontend**: Bootstrap 5, Plotly.js
- **Database**: SQLite (development), PostgreSQL (production)
